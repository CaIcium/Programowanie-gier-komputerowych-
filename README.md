# Programowanie-gier-komputerowych- LAB4
## Co zostało zrobione:
Pierwszym krokiem było podział kodu na funkcjie w celu posegregowania kodu, ulepszenia przejrzystości kodu. Następnie dodajemy opcję menu w momęnice włączenia gry, gdzie jak klikniemy spacje to zaczyna się gra, kiedy klikniemy klawisz L to wyświetla się ranking i na końcu w menu wyświetlamy najwyższy score w rankigu. Kolejną rzeczą jest dodanie opcji zapisu w rankigu wyniku wraz z podaniem przez gracza inicjałów, po tym wyświetla się menu z możliwością restu albo powrotu do menu. W grze jeszce dodałem wyświetlanie aktualnego wyniku i ilość żyć gracza, kiedy ilość spadnie na 0 to przegrywamy albo kiedy rój zostanie znisczony to wygrywamy.
