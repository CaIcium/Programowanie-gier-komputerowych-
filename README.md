# Programowanie-gier-komputerowych- LAB3
## Co zostało wykonane:
W tej części zajęć trzeba było sprawić aby duszek poruszany przez gracza mógł strzelać pociskami. W tym celu sprawiłem, że na ekranie może być maksymalnie 3 pociski gracza, ponieważ w przeciwnym przypadku jakby nie miał limitów pocisków to za każdym wstrzelniem optymalizacja gry by spadała, plus powstawały inne błędy logiczne. Nasępnie trzeba było zrobić kloizje między obiektami, czyli w przypadku kiedy linia wrogra doleci do statku to przegrywamy, albo kiedy pocisk gracza zostanie wystrzelony w preciwnika to powstaje kolizja i przeciwnik zostaje zneutralizowany i vice versa. Dodatkowo jeszcze dodano ogień nieprzyjaciela.
