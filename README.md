# Programowanie-gier-komputerowych- LAB6
## Co zostało wykonane:
Na początek trzeba było zrobić płynne przechodznie statku z krawędzi, to akurat zrobiłem juz w poprzednim labie (lab_05), następnie utworzyłem klase asteroid.py, która odpowiada za ruch, kształt, rysowanie. W main ustwaiam liczbę asteroid. Następnie musimy dodać coś w stylu przejścia astridy podobnie jak z statkiem. Do przetestowania umiesciłem asterode na środku krawiędzi ekranu. Na koniec wydzieliłem logikie przejść do oddzielnego pliku utils.py.
## Uruchomienie:
Aby uruchomić program trzeba pobrać z tego katalogu skrypty main.py, statek.py, asteroid.py i utils.py następnie najlepiej umiescić je w jednym katalogu, następnie odpalić katalog w np. visual studio code i skomplować plik main.py.
