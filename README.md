# Programowanie-gier-komputerowych- LAB2
Pierwszym korkiem była definicja poziomu, musiałem zdefinować mapę roju. Następnie po utworzeniu mapy tworzę pustą tablice i wypełniam przeciwnikiami. Wcześniej jeszcze trzeba było utworzyć kilka modeli spritów, aby dałoby się je wyświetlić na mapie. Np 1 = sprite o nazwie 1. Następnie w funkcij draw() musiałem wyświetlić rój. Trzebabyło jescze dodać ruch roju wraz z dodaniem prędkości roju i odbicie roju z prawej na lewą i lewą na prawą strone w momęcie kolizji z końcami ekranu.
