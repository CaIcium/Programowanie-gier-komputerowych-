# Programowanie-gier-komputerowych- LAB5
## Co zostało zrobione:
Na początek utworzyłem puste okno z czarym tłem i klasę statek z konstruktorem przyjmującym pozycję startową. Następnie rysujemy statek, jest to po prostu trujkąt następnie dodaje rotację statku. Kolejnym krokiem jest możliwość poruszania się staku w prawo, lewo i w kierunek gdzie jest skierowany dzób statku. Do prouszania się statku doałem jeszcze tarcie i limit prędkościu statku. W każdej klatce zmniejszana jest prędkość statku kiedy przycisk gazu nie jest wciskany. Dodatkowo dodałem, że statek przechodzi z prawo na lewo i z góry na dół vice versa, w momenice kiedy trafia na koniec ekranu programu.
## Uruchomienie:
Aby uruchomić program trzeba pobrać z tego katalogu skrypty main.py i statek.py następnie najlepiej umiescić je w jednym katalogu, następnie odpalić katalog w np. visual studio code i skomplować plik main.py.
